#Team Management System

## Description
A backend system designed for managing professional sports rosters and financial data. The project focuses on object-oriented programming (OOP) principles, efficient data handling, and robust persistence using Java Serialization.

## Key Features
* **Data Persistence**: Utilizes Java Serialization to save and load team data from binary files.
* **Complex Data Management**: Uses `HashMap` and `ArrayList` for efficient data storage and `EnumSet` for multi-position player support.
* **Data Integrity**: Implemented strict validation and custom exception handling to prevent invalid inputs (e.g., negative salaries or ratings).
* **Fail-fast Principle**: Designed for runtime error detection to ensure system stability.

## Technologies Used
* **Language**: Java Core
* **Concepts**: OOP (SOLID, DRY), Clean Code
* **Data Structures**: Collections Framework (`HashMap`, `ArrayList`, `EnumSet`)
* **I/O**: Java Serialization (ObjectOutputStream/ObjectInputStream)
